# Meme Sound Pad

Autores:
- Adrien Bezerra Leandro
- Felipe Araujo Carvalho

Este é um projeto simples desenvolvido em Java Swing que simula uma mesa de som de memes.

O aplicativo apresenta uma interface com 9 botões organizados em uma grade. Cada botão está associado a uma imagem e a um arquivo de áudio (.wav). Ao clicar em um botão, o som é reproduzido em uma Thread separada para garantir que a interface não trave, e o botão altera seu visual temporariamente para indicar que o áudio está tocando.

Nessa aplicação é possível tocar mais de um áudio simultâneamente.
